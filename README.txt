This documentation is built with Python and mkdocs.
If you want to update it, follow these steps:

1. Clone this repository
2. Create a virtual environment
3. Install the project requirements from `requirements.txt`
4. Read the quickstart guide to learn how to use mkdocs: https://www.mkdocs.org/
5. Make your changes to the doc files, commit and push to GitHub
6. Deploy by running `mkdocs gh-deploy`

The live documentation will be updated in a couple of minutes.
