This documentation is built with Python and mkdocs.
If you want to update it, follow these steps:

1. Clone this repository
2. Create a virtual environment (ex. python3 -m venv .venv)
3. Activate venv (ex. source .venv/bin/activate)
4. Install the project requirements from `requirements.txt` (ex. pip install -r requirements.txt)
5. Exclude .venv from git (add to .gitignore)
6. Read the quickstart guide to learn how to use mkdocs: https://www.mkdocs.org/
7. Make your changes to the doc files, commit and push to GitHub
8. Deploy by running `mkdocs gh-deploy`

The live documentation will be updated in a couple of minutes.
